# 漢字マスター (Kanji Master)

小学生向けの漢字学習アプリケーションです。1年生から6年生で習う漢字を、「読み」と「書き」の両面からテスト形式で学習できます。

## 特徴

- **学年別学習**: 小学1年生から6年生までの配当漢字を網羅。
- **読みテスト**: 漢字の正しい読み方を解答。
- **書きテスト**: 提示された読みから、正しい漢字を思い出す練習。
- **履歴機能**: 学習の進捗や正答率を可視化。
- **弱点リスト**: 苦手な漢字を自動的にリストアップし、重点的に学習可能。

## インストール・実行方法

このプロジェクトは静的HTML/JavaScriptで構成されているため、特別なサーバー設定なしで実行可能です。

1.  このリポジトリをクローンまたはダウンロードします。
2.  `index.html` をブラウザで開きます。

漢字データは `js/data-grade*.js` に格納されています。修正を行った際は、`check_missing_readings.ps1` などのスクリプトを使用して整合性の確認を行ってください。
